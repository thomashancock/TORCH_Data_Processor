Detailed Description

template<unsigned int N>
class ErrorCounter< N >

Used by ErrorSpy to track which readout combinations of board/tdc ids that errors have occured on.

Template class.

Template parameter N is the dimensionality of the ErrorCounts to be stored i.e. For errors specific to a board/tdc id combination, N = 2. For generic errors, N = 0.

N is given as a parameter to facilitate easier modifications to the ErrorSpy in the future.
